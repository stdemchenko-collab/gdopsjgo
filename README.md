# Online Store - HTML версия 🛍️

Полнофункциональный интернет-магазин, написанный на чистом HTML, CSS и JavaScript. Отлично подходит для быстрого запуска e-commerce сайта без необходимости в сложных фреймворках и серверных технологиях.

## Возможности

✅ **Каталог товаров** - сетка товаров с фильтрацией по категориям  
✅ **Поиск товаров** - быстрый поиск по названию и описанию  
✅ **Страницы товаров** - детальное описание каждого товара  
✅ **Похожие товары** - рекомендации похожих товаров  
✅ **Корзина покупок** - полная функциональность корзины с localStorage  
✅ **Изменение количества** - легкое управление количеством товаров  
✅ **Оформление заказа** - простой checkout процесс  
✅ **Адаптивный дизайн** - отлично работает на мобильных устройствах

## Как запустить

### Способ 1: Простой веб-сервер (Python)
```bash
# Если установлен Python 3
python -m http.server 8000

# Если установлен Python 2
python -m SimpleHTTPServer 8000
```

Затем откройте в браузере: `http://localhost:8000`

### Способ 2: Live Server (VS Code)
1. Установите расширение "Live Server" в VS Code
2. Кликните правой кнопкой на `index.html`
3. Выберите "Open with Live Server"

### Способ 3: Node.js (http-server)
```bash
npm install -g http-server
http-server
```

## Технические характеристики

### Хранилище данных
- Товары хранятся в `data.js` (12 образцов товаров)
- Корзина сохраняется в `localStorage` браузера
- Данные сохраняются между загрузками страницы

### Функциональность

**Фильтрация:**
- По категориям на главной странице
- Кнопки фильтра для быстрого выбора

**Поиск:**
- По названию товара
- По описанию
- Поддержка Enter-клавиши

**Корзина:**
- Добавление/удаление товаров
- Изменение количества
- Расчет стоимости доставки (бесплатно при заказе > 5000₽)
- Расчет итоговой суммы

## Цены в примере

Используется валюта - российские рубли (₽). Цены из примера:

- Электроника: 1,299₽ - 119,999₽

- **300₽** при заказе меньше 5,000₽

## Кастомизация

### Добавить новый товар
Откройте `data.js` и добавьте объект в массив `products`:

```javascript
{
    id: 13,                          // Уникальный ID
    name: "Название товара",         // Название
    category: "electronics",         // Категория (electronics, books, clothes)
    price: 9999,                     // Цена в рублях
    image: "📱",                     // Эмодзи или символ
    rating: 4.5,                     // Рейтинг 1-5
    description: "Описание",         // Краткое описание
    details: "Подробные характеристики..." // Детальное описание
}
```

### Изменить цвета
Все цвета в переменных CSS в `styles.css`:

```css
--primary-color: #2563eb;      /* Основной синий */
--secondary-color: #1e40af;    /* Темный синий */
--accent-color: #f59e0b;       /* Оранжевый */
--success-color: #10b981;      /* Зелёный */
--danger-color: #ef4444;       /* Красный */
```

### Добавить новую категорию
1. Добавьте категорию в свойство `category` товара в `data.js`
2. Кнопка фильтра автоматически появится... нет, нужно также обновить фильтр-кнопки в `index.html`
